      "rsi_4h": <float>,

      "volume_24h": <float USDC>,
      "signal_1d_rate_limited": <bool — True si l'analyse 1D a échoué par rate limit>,
      "in_portfolio": <bool — True si le coin a une position open dans trade_history>
    },
    ...
  },
  "top_gainers_symbols": [...],
  "breakout_symbols": [...],
  "sentiment": "Bullish|Neutral|Bearish",
  "open_positions": <int>,
  "budget_disponible": <float USDC>,
  "config": null
}

RÈGLES pour construire analysis_results :
- Utilise UNIQUEMENT les données produites par Phase 2 (coin_analysis 4h, coin_analysis 1D si disponible)
- signal_1d_rate_limited = True si et seulement si l'appel coin_analysis 1D a retourné une erreur de rate limit ou a échoué
- signal_1d = null si l'analyse 1D n'a pas été tentée ou a échoué
- in_portfolio = True si le coin apparaît dans trade_history avec status="open"
- NE PAS inventer, extrapoler ou hardcoder des valeurs — copier les données exactes de Phase 2
